qfa: A set of tools for carrying out Quantitative Fitness Analysis

Introduction

In the qfa method, we generate cell density timecourse estimates by repeatedly photographing micro-organism cultures growing in arrays on solid agar plates. We quantify cell density using the image analysis tool Colonyzer (Lawless et al., 2010). We fit a logistic model to each timecourse for each independently growing culture, and use the logistic model parameters to define a quantitative measure of culture fitness.

Applications

If the arrayed cultures contain relevant genetic mutations, we can use fitness estimates to infer the presence or absence of epistasis, and quantify genetic interaction strengths. If the arrayed cultures are grown at selected temperatures we can infer the extent of temperature sensitivity of mutants. We have described both of the above applications in Addinall et al., 2011. If cultures are subject to treatments (e.g. nutrition, or drug treatments) we can generate ranked lists of mutations affected by the treatment.
